# Day 05: HTML & CSS Basics Learning

## Overview
Day 05 focuses on fundamental HTML and CSS concepts including pseudo-classes, image galleries, navbar creation, and responsive design patterns.

## Learning Topics

### 1. **Pseudo Classes in CSS**
- Understanding CSS pseudo-classes (:hover, :active, :focus, :visited, etc.)
- Applying pseudo-classes for interactive elements
- Styling links and form elements

### 2. **Image Galleries**
- Creating responsive image galleries
- Flexbox-based layouts for image organization
- Image styling and hover effects
- Multiple categories/sections management

### 3. **Navbar/Navigation Design**
- Creating sticky/fixed navigation bars
- Logo placement and styling
- Navigation links layout
- Search functionality integration

### 4. **2D CSS Transformations**
- Understanding transform property
- Rotation, scaling, and translation
- Creating visual effects on images
