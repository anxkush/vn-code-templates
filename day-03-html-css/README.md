# Day 03 - HTML & CSS Learning

## Projects Included:

### 1. WEBSITE.HTML
- Main portfolio/website project
- Features: Navigation menu, sections, article content
- Includes: Profile image, color selector dropdown
- Content: Website structure with header, nav, and main sections

### 2. EXCEL.HTML
- University records table
- Displays student data with roll numbers and university info
- Table structure with colspan

### 3. EXCEL-OWN.HTML
- Iris dataset table
- Flower measurements (Sepal and Petal lengths/widths)
- Structured table with numerical data

### 4. EXCEL-ROW.HTML
- Email credentials table
- Website, email address, and password columns
- Uses rowspan for merged cells

### 5. WEBSITE.CSS
- Styling for the main website
- Flexbox layout implementation
- Border radius, colors, and spacing properties
- Navigation and button styling

## Key Concepts Covered:
- HTML table structure and attributes (colspan, rowspan)
- Navigation and semantic HTML
- CSS Flexbox layout
- Form elements (select, option)
- CSS styling (colors, borders, padding, margins)
- Image embedding and responsive sizing
