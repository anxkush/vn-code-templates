# Day 06: Learning HTML and CSS

## Overview
This folder contains a collection of HTML and CSS learning projects covering various web design concepts and techniques.

## Projects Included

### 1. **index.html** - Main Landing Page
- Central overview page showcasing all Day 06 projects
- Navigation links to all sub-projects
- Demonstrates semantic HTML structure

### 2. **animation.html** - CSS Animations & Transitions
- Explores CSS animation techniques
- Interactive elements with smooth transitions
- Foundation for understanding motion in web design

### 3. **gallery.html** - Photography Gallery
- Responsive image grid layout
- Hover effects and smooth animations
- Image captions and interactive elements
- Demonstrates CSS Grid/Flexbox responsiveness

### 4. **resume.html** - Professional Resume Website
- Personal resume with professional layout
- Navigation menu with anchor links
- Multiple sections: About, Skills, Experience, Education, Projects
- Animations using CSS classes (fade-in, slide-in-left, etc.)
- Print-friendly design with print button

### 5. **restaurant.html** - Restaurant Landing Page
- Hero section with call-to-action
- Menu highlights section
- Contact/Reservation form
- Clean, professional restaurant business layout

## Key Learning Concepts

- **Semantic HTML**: Proper use of HTML5 tags
- **Responsive Design**: Mobile-friendly layouts
- **CSS Animations**: Transitions and keyframe animations
- **Forms**: User input with form elements
- **Typography**: Font integration and styling
- **Color & Design**: Professional color schemes
- **Accessibility**: Semantic markup and alt text
